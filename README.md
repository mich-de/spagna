# Sol & Local — Costa del Sol 2026

Dashboard di viaggio premium per la Costa del Sol (19-25 Giugno 2026).

## Caratteristiche
- 🏖 Itinerario dettagliato giorno per giorno (spostato in pagina dedicata `/itinerary`).
- ⛵ Giri in Barca & Catamarani (BoatTours) nella zona di Nerja e Maro.
- 🛍 Mercati & Rastro settimanali della Costa del Sol.
- 🔥 Speciale Noche de San Juan 2026.
- 🍽 Ristoranti e Chiringuitos autentici verificati.
- 🚗 Logistica e guida ai parcheggi.
- 💰 Gestione budget su 3 livelli.
- 🏖 Costa de Nerja & Maro: spiagge selvagge, paella cotta a legna e attrazioni della costa orientale.
- ✨ Centro Ispirazione: Deck interattivo per idee al volo e Matcher per generare la "Giornata Perfetta" in una zona specifica salvandola nel piano di viaggio con 1 clic.
- 🍹 Movida Over 35 & Strategia Dating: Guida operativa e mappa strategica della Costa del Sol (Málaga, San Pedro, Marbella, Estepona, Nerja, Benalmádena) orientata alla socializzazione locale e integrazione di link per le app di dating.
- 🔒 Password Wall client-side (con hash SHA-256) per proteggere la riservatezza dell'app in ambienti statici (GitHub Pages).

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS v4
- Framer Motion

## Installazione Locale
```bash
npm install
npm run dev
```
Oppure usa `start.bat` su Windows.

## Deployment
Il sito è configurato per il deploy automatico su **GitHub Pages** tramite GitHub Actions.
